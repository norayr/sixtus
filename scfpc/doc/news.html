Seksi Commander - news

25.4.2003 - alpha version of Lazarus port

  • The most important change is port to FPC and Lazarus. You do not need libbor*.so anymore in the new version. Original Kylix version is no longer maintained. Main development is now done only with FPC version.
  • Added compare files dialog (visual compare side by side like in Total Commander) for text files. Comparing is in beta stage but very usable
  • Fully configurable icon support (you can specify xpm icon for any kind of file)
  • Fixed space counting problem
  • Viewer and Editor can now have multiple instances at one time
  • Fixed problem with deadkeys (thanks to lazarus team)
  • Due to porting some keyboard shortcuts do not work (report this and send me (or try to fix) which hotkeys are missing (I think that you should be able to fully control the program with just keyboard))
  • Don't forget sc.ext where you can specify colors for any kind of file in panel
  • Running external commands on files in viewer is temporarily (I hope) disabled
  • VFS support is disabled (but interface is done)
  • Font problems are worked on (try and see)
  • You can specify display style of filenames in the configuration dialog (separate extension?)
  • See complete changelog.txt
