Seksi Commander - How compile Seksi Commander

What you need?

At first must download FreePascal. I use FPC 1.9.3 (2.0 beta3). In some distribution is FreePascal included (for example in MDK in fpc package), but this is stable version.

After this download Lazarus. Seksi Commander use Lazarus from CVS.

Now this download Seksi Commander source code. In downloaded file is included also viewercontrol, so install it to the Lazarus. Now open file sc.lpi or scgtk2.lpi. In second case you must also download gtk2forpascal from source forge. Warning: gtk2 version of seksi commander not work properly (editor crash and movement in panels is slow).

Running Seksi Commander

Simple go to directory with Seksi Commander and ./sc. All files are stored in this directory. In final version will be in home directory.

Some hints

If you uses Lazarus with XFce disable CTRL+SHIFT UP, DOWN (move between declaration and implementation) in configuration of windows manager.
