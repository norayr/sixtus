Seksi Commander homepage

Seksi Commander is a filemanager for Linux. It is strongly inspired by Total Commander (formerly Windows Commander), Midnight Commander and features some new ideas.
Features: internal viewer (binary, hexadecimal, text mode, image view), internal editor with syntax highlighting, internal visual differ (all of these can be replaced by external ones), MultiRename tool and more.

Since version 0.6alpha SC is written in Lazarus and FreePascal and uses gtk with user themes.
